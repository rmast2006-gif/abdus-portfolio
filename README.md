🚀 Abdus Portfolio

A Full Stack Portfolio Website built using the MERN Stack, featuring an Admin Dashboard, Image Upload, and Dynamic Content Management. This project allows you to showcase projects, manage content, and update your portfolio in real-time.

🔗 Live Demo: https://abdus-portfolio.vercel.app/

📌 Features
✨ Modern and responsive UI
🔐 Admin Dashboard for content management
🖼️ Image upload functionality
📂 Dynamic project and content handling
⚡ Fast performance with optimized frontend
🌐 Fully deployed (Frontend + Backend)
🛠️ Tech Stack

Frontend:

React.js
Tailwind CSS
Framer Motion

Backend:

Node.js
Express.js

Database:

MongoDB

Other Tools:

Cloudinary (for image uploads)
Vercel (deployment)
